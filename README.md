## ✨ Features

- 🌍 Search weather by city
- 🌡️ Real-time temperature
- ☁️ Weather conditions
- 💨 Wind speed
- 💧 Humidity
- 📱 Fully responsive design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
